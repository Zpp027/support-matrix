## 安装步骤

### 刷写镜像

使用 `unzstd` 解压镜像。
清空你的 sd 卡。
使用 `dd` 将镜像写入 microSD 卡。

```bash
unzstd /path/to/fedora.raw.zst
sudo wipefs -a /dev/your_device
sudo dd if=/path/to/fedora.raw of=/dev/your_device bs=1M status=progress
```

### 登录系统

*系统启动较为缓慢。*

通过串口登录系统。

默认用户名： `root`
默认密码： `riscv`
